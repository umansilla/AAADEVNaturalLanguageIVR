Create Verbio User

If a Verbio user is not found associated with the current account, the button to create a user in Verbio will be depleted.

  1. Click on the Create Verbio User button
  2. The user's name in Verbio will be created automatically. after clicking the Create Verbio User button.
  3. Assign a phone number from which the call will be made, it is important, because we must have a phone associated with the user's account.
  4. Click on the Save button, located just below the button to create the user in Verbio. or click Cancel to not create the user.

Recording controls

Once you have a Verbio user and a telephone associated with the account, it is necessary to start recording your voice.

  1. Click on the Record button and allow the browser to use the local microphone.
  2. In the browser tab you will notice a red circle, which indicates that you are starting the recording
  3. Mention the following phrase "In Avaya my voice is my password". mention the phrase according to the language you want.
  4. Click on the Stop button. You will notice that a temporary audio file has been created in the left column, below the recording controls.

Note: Remember that Verbio does not save data until the training is completed

Warning: in the process of saving do not refresh the screen without having saved the audio files. The recorded files (without saving) are temporary.

As soon as you have several audio recorded with which you are satisfied. It's time to add the audio to Verbio.

Each temporary audio contains 3 buttons.

Start Adding files

  1. Download : Allows you to download the corresponding audio to the local storage
  2. Analize with Verbio BioMetrics : Send the audio to Verbio to find the result according to the Verbio analysis, where 0 is the lowest value and 100 is the maximum value. (To analyze an audio it is necessary that the field Trained biometrics indicate Yes, otherwise the tool will show Error).
  3. Add File To Verbio : It is the first step that must be carried out to start training in Verbio. It is recommended to add at least 10 audios to Verbio. When adding an audio, the table on the right side with name "Uploaded to Verbio" will be updated.

As soon as you have the first file in the "Uploaded to Verbio" table, the button to train the tool will appear. Pressing the button allows the Verbio user to be enabled to perform the voice analysis. It is recommended to press the button each time several new recordings are added.
