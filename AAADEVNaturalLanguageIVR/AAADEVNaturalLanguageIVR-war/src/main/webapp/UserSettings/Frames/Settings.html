Settings Page Guide

In the settings section it is possible to update the user data.

  1. The name: This name will be reproduced at the moment the Verbio BioMetrics tool recognizes the voice.
  2. The telephone: it is important to have registered the telephone from which the call will be made to Watson, because the database identifies if it is a registered user through the telephone. Important! Add the phone starting with "+" for example +5215555555555, without spaces.
  3. The country: updating the country is necessary so we know if the international key of the phone is correct.
  4. Password: The password field updates the current password, when you log out you must enter the new password.

Press the Save button to modify the fields that have text. It is not necessary to have all the fields filled.

The Reset button will erase all the information that has been put in the fields of the form.
